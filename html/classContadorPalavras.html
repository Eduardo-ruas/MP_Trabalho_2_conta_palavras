Detailed Description

Classe responsável por contar a frequência de palavras em um arquivo de texto.

Esta classe fornece um método para ler um arquivo, contar a frequência de palavras diferentes e retornar um mapa com as palavras em ordem alfabética e suas respectivas ocorrências.

std::map< std::string, int > ContadorPalavras::contarPalavras ( const std::string &  nomeArquivo)

Conta a frequência de palavras em um arquivo texto.

Conta a frequência de cada palavra em um arquivo de texto.

Este método lê o arquivo linha por linha, remove pontuações, converte as palavras para letras minúsculas e calcula a frequência de cada palavra. As palavras são organizadas em ordem alfabética no mapa de retorno.

Parameters
nomeArquivoO caminho ou nome do arquivo a ser lido.
Returns
Um mapa onde a chave é a palavra e o valor é a quantidade de ocorrências. Retorna um mapa vazio se o arquivo não puder ser aberto.
